Project Name: AI Ecosystem Coach

Product Summary:
AI Ecosystem Coach is a multi-agent chatbot that helps users design, evaluate, and care for gardens or ecosystems using grounded, personalized AI guidance. The system combines an LLM Coach, specialized agents, retrieval-augmented generation, persistent memory, and a decision engine to provide actionable ecosystem recommendations.

Core Problems Solved:
- users do not know what plants fit their conditions
- users struggle to evaluate sustainability and biodiversity
- users have difficulty diagnosing plant problems
- users need personalized guidance that improves over time

Primary Users:
- home gardeners
- beginner plant owners
- sustainable landscaping enthusiasts
- pollinator and native plant gardeners
- educators and community garden users

Main User Journeys:
1. ecosystem planning
2. sustainability evaluation
3. biodiversity improvement
4. plant diagnosis
5. personalized follow-up guidance

MVP Features:
- natural-language chat interface
- Orchestrator Agent
- Plant Planning Agent
- Sustainability Agent
- Wildlife / Biodiversity Agent
- Diagnosis Agent
- Memory Agent
- RAG layer
- persistent database and memory
- decision engine
- structured response format

Functional Requirements:
- recommend plants based on conditions
- evaluate sustainability
- score biodiversity support
- diagnose plant problems from symptoms
- retrieve grounded ecosystem knowledge
- remember user preferences and prior outcomes
- merge specialized agent outputs into one final response

Non-Functional Requirements:
- grounded and accurate
- explainable
- modular
- scalable
- responsive
- persistent
- safe and practical

System Goals:
- personalized ecosystem guidance
- grounded recommendations
- practical next steps
- improved sustainability and biodiversity outcomes
- support for long-term ecosystem improvement
